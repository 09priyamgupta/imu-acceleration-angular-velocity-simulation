# IMU Acceleration & Angular Velocity Simulation

This project simulates IMU sensor outputs for a vehicle motion scenario involving:

- Acceleration (0 to 20 m/s)
- Straight-line travel at constant speed
- A 100 m radius U-turn at constant speed
- Emergency braking to stop in 1 meter

The notebook computes and visualizes:

- Longitudinal acceleration $a_x(t)$
- Lateral acceleration $a_y(t)$
- Angular velocity $\omega(t)$

Files:
- `imu_acc_angular_vel.ipynb` — Main Jupyter Notebook
- `problem_statement.png` — Scenario diagram
